#ONLYOFFICE DocumentServer no Kubernetes
======

Este serviço foi construído a partir da conversão com o [Kompose](https://github.com/kubernetes-incubator/kompose) do arquivo [docker_compose](https://github.com/ONLYOFFICE/Docker-DocumentServer/blob/master/docker-compose.yml) do ONLYOFFICE. Também utilizamos [Running Document Server using Docker Compose](http://helpcenter.onlyoffice.com/server/docker/document/docker-compose.aspx) e [Installing Document Server for Docker on a local server](http://helpcenter.onlyoffice.com/server/docker/document/docker-installation.aspx).
